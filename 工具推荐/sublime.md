## 快捷键速记

```
Alt + shift + 2,5,8 #分屏大法
Ctrl + shift + p #语言设置 
Ctrl + p #查找文件
Ctrl+Shift+L #分散编辑
Ctrl + J #合并
Ctrl + Enter #向下插入一行
Ctrl + Shift + Enter #向上插入一行
Ctrl + Shift + ↑/↓ #行上下移动
Shift + 鼠标右键 #块选择
Ctrl + H #查找和替换
Ctrl + D #单词选中
Ctrl + shift + D #复制行
Ctrl + shift + K #删除行
Ctrl + shift + F #全文档搜索，可能会和搜狗输入法快捷键冲突
```
